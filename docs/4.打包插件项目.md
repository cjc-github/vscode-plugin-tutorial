# VScode插件打包

## 1. 安装vsce工具

要打包 VSCode 插件，首先需要安装 vsce 工具，这是 Visual Studio Code Extension Manager (VSCE) 的命令行工具，用于打包、发布和管理 VSCode 插件。

```bash
npm install -g vsce
```

## 2. 打包插件

在插件项目的根目录下运行以下命令，这将会生成一个.vsix文件, 这个就是插件的安装包。

```bash
vsce package
```

## 3. 安装插件

使用以下命令安装打包好的插件：

```bash
code --install-extension ./your-extension-0.0.1.vsix
```
